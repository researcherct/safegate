Abstract

Large Language Models (LLMs) are increasingly used to plan and generate robot actions from natural language commands. However, current safety approaches for LLM-based robot systems suffer from critical limitations: binary safety judgments with no deferral mechanism, absence of formal safety contracts, single-pass reasoning vulnerable to hallucination, and context-blind evaluation that ignores scene and user factors.

We present SafeGate, a neurosymbolic pre-execution safety gate that combines LLM-based hazard identification with deterministic safety logic and formal task safety contracts. SafeGate introduces a three-decision framework (AUTHORIZE / DEFER / REJECT) that enables the system to request clarification for ambiguous tasks rather than making potentially dangerous assumptions. Our multi-stage pipeline includes consistency-checked hazard identification, canonical hazard template matching, deterministic gate rules, and a verification layer that cross-checks decisions.

We evaluate SafeGate on a benchmark of 230 robot tasks spanning safe, unsafe, and ambiguous scenarios across diverse environments. SafeGate achieves a 100% safety rate (blocking all unsafe tasks) while maintaining 85%+ feasibility (authorizing safe tasks), significantly outperforming baseline approaches including direct LLM prompting (SELP) and rule-based systems (RoboGuard).

System Architecture

Key Innovations

Three-Decision Framework: Unlike binary safe/unsafe approaches, SafeGate introduces DEFER — requesting clarification for ambiguous tasks (e.g., "bring hot coffee to my daughter" — how old is the daughter?).

Consistency Voting: Hazard identification runs 3 times with majority voting to filter LLM hallucinations. Only hazards confirmed by 2+ runs are retained.

Task Safety Contracts: Each authorized task receives a formal safety contract with preconditions, postconditions, invariants, guards, and abort conditions that can be monitored during execution.

Scene-Aware Capabilities: Robot capability profiles are dynamically adjusted based on scene context — a robot in a warehouse is assumed to handle heavy loads; the same command in a nursery triggers stricter safety checks.

Real-World Experiments

SafeGate is deployed on physical robots including a 4-DOF table-top robot arm (Raccoon) with USB camera for vision-guided manipulation, and the Hamster-S mobile robot with gripper. The system performs camera-based scene understanding, safety validation, and code generation for real-world task execution.
